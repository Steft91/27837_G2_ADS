# Biblioteca de Soporte

Esta carpeta contiene documentos auxiliares, de referencia y material de consulta que sirven de apoyo al proyecto **Sistema de Préstamo de Proyectores ESPE**.

## Propósito

La Biblioteca de Soporte almacena artefactos que no forman parte directa de los entregables principales, pero que son necesarios para el desarrollo del proyecto, como plantillas, guías, estándares, material de capacitación y documentación de referencia.

## Estructura

### 1. ELICITACIÓN
- Plantillas para documentación de requisitos
- Guías de entrevistas y encuestas
- Formatos de historias de usuario
- Checklist de validación de requisitos
- Material de referencia sobre técnicas de elicitación

### 2. DISEÑOS
- Plantillas de diagramas UML
- Guías de estilo de código
- Estándares de diseño de interfaz
- Paleta de colores institucional ESPE
- Especificaciones de accesibilidad
- Mockups de referencia
