# Biblioteca Maestra

Esta carpeta contiene las versiones **aprobadas y finales** de los documentos del proyecto **Sistema de Préstamo de Proyectores ESPE**.

## Propósito

La Biblioteca Maestra almacena los artefactos que han completado el ciclo de revisión, validación y aprobación, constituyendo las **líneas base** oficiales del proyecto. Estos documentos son de solo lectura y representan el estado formal del proyecto en momentos específicos.

## Control de Versiones

Cada documento en la Biblioteca Maestra sigue el formato de versionado:

- **vX.Y** donde:
  - X = Versión mayor (cambios significativos)
  - Y = Versión menor (ajustes y mejoras)

Ejemplo: `SRS_v1.2.pdf`

## Proceso de Promoción

Un documento llega a la Biblioteca Maestra mediante:

1. **Desarrollo** en Biblioteca de Trabajo
2. **Revisión** por el equipo
3. **Validación** con stakeholders
4. **Aprobación** formal del profesor/coordinador
5. **Promoción** a Biblioteca Maestra como línea base

## Acceso y Modificación

⚠️ **IMPORTANTE:**
- Los documentos en esta biblioteca son **de solo lectura**
- Cualquier modificación requiere:
  1. Crear una solicitud de cambio formal
  2. Trabajar en copia en Biblioteca de Trabajo
  3. Pasar por proceso de revisión y aprobación
  4. Promocionar como nueva versión a Biblioteca Maestra
- Se mantiene historial completo de todas las versiones
