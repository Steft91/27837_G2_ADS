# Biblioteca de Trabajo

Esta carpeta contiene todos los documentos que están actualmente en desarrollo o revisión para el proyecto **Sistema de Préstamo de Proyectores ESPE**.

## Propósito

La Biblioteca de Trabajo es el espacio donde se gestionan los artefactos en proceso de elaboración, validación y refinamiento antes de ser promovidos a la Biblioteca Maestra como versiones aprobadas.

## Flujo de Trabajo

1. **Creación:** Los documentos se crean inicialmente en esta biblioteca
2. **Revisión:** Se revisan y validan con stakeholders
3. **Refinamiento:** Se realizan ajustes según feedback
4. **Aprobación:** Una vez aprobados, pasan a Biblioteca Maestra
5. **Versionado:** Se mantiene control de versiones mediante Git y Jira

## Gestión de Cambios

Cualquier solicitud de cambio debe seguir el proceso definido en el Plan de Gestión de Requisitos:

1. Propuesta del cambio
2. Evaluación del impacto
3. Aprobación o rechazo
4. Implementación y seguimiento

Tipos de cambio:
- **Correctivo:** Errores o ambigüedades
- **Adaptativo:** Nuevas políticas o requerimientos
- **Excepcional (Hotfix):** Críticos que requieren atención inmediata
